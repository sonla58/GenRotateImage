## Description

A small tool to generate 0 to 360 rotated image from origin image :D

## Installation

```
npm install as-gen-image
```

## Usage

```
GenImage rotate <file_path> -s <scale> -d <distance> -c <color>
```

**Example**

```Command
GenImage rotate Images/Test.png -s 2 -d 10 -c '#FFF'
```
